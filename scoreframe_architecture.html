⚡ Key Data Flows

Camera → GL Pipeline SurfaceTexture
GL Pipeline → Display Surface 1
GL Pipeline → Recording Surface 2 → MediaCodec → MP4
GL Pipeline → Stream Surface 3 → StreamEncoder → RTMP
Scoreboard → GL Pipeline Canvas → Bitmap → GL Texture
Score Change → Scoreboard ScoreboardType → Renderer → Bitmap
Remote Client ⇄ Remote Server TCP / Wi-Fi Direct
RTMP Client → YouTube / IVS RTMP over TCP
Team/Game Data ⇄ Firestore ScoreFrameSync.kt
Sign-In → Firebase Auth Credential Manager → ID Token
Score Event → IVS Metadata PutMetadata (Sig V4)
Highlight ⭐ → Clip Extraction Timestamp → PostProcessing → MP4

🎬 Triple-Surface Render Pipeline

Camera2 → SurfaceTexture Hardware frames
SurfaceTexture → GL External Texture OES_EGL_image
ScoreboardRenderer → Bitmap Canvas draw (12 sports)
Bitmap → GL Texture 2D glTexImage2D
CameraRenderer → Composite Frame Camera + Scoreboard + Watermark + PiP
Composite → Surface 1: SurfaceView Live preview on screen
Composite → Surface 2: MediaCodec → MP4 file (if recording)
Composite → Surface 3: StreamEncoder → RTMP (if streaming)
▸ Firestore Data Structure

📁 users/{userId}/

├── profile — displayName, email, createdAt, lastSyncAt, tier
├── rosters/{sport} — teamName, color, players[], battingOrder[]
├── teams/{teamId} — name, sport, type, color, roster[], isSynced
├── sportConfigs/{sport} — match format, rules preferences
├── games/{gameId} — sport, date, teams, scores, duration, participants[]
│   └── scorebook — full baseball play-by-play (subcollection)
├── stats/{sport} — career stats aggregated per sport
├── players/{playerId} — bowling avg/high, batting avg, W-L records
├── teamStats/{teamId} — W/L/T, pointsFor, pointsAgainst, gamesPlayed
└── marketing_subscribers/{userId} — email, optedIn, subscribedAt

📱 Local Storage (filesDir/)

├── teams/{id}.json — SavedTeam profiles (TeamManager)
├── game_history/{id}.json — GameSummary records
├── game_scorebooks/{id}.json — BaseballFullGame companion files
├── player_stats/{id}.json — PlayerCareerStats per player
├── team_stats/{id}.json — TeamCareerStats per team
├── career_stats.json — aggregated per-sport career stats
├── baseball_game_state.json — active scorebook (auto-save)
└── SharedPreferences — app settings, scoreboard state, session count
